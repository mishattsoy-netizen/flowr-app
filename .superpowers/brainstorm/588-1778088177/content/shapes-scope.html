Shape Tools — which ones?

Excalidraw has several drawing primitives. Which do you want?

A

Rectangle / Square

Click-drag to draw. Shift to constrain to square.

B

Ellipse / Circle

Click-drag to draw. Shift to constrain to circle.

C

Diamond

Rotated square — common in flowcharts for decisions.

D

Arrow / Line

Click-drag to draw directional arrows or plain lines between points.

E

Freedraw (pen tool)

Freehand strokes with pressure-sensitive width.

F

Text

Click to place a text element, double-click to edit inline.

Select all you want, then reply in the terminal.
