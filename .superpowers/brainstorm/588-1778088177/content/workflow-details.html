Workflow — which interaction patterns?

These are the key Excalidraw workflow features. Pick what you want.

A

Multi-select with drag box

Click and drag on empty canvas to rubber-band select multiple elements

B

Grouping (Ctrl+G)

Group selected elements so they move/resize as one unit

C

Style panel

Fill color, stroke color, stroke width, opacity — shown when element is selected

D

Snap to grid / snap to objects

Elements snap to a grid or to edges of other elements while dragging

E

Undo / Redo (Ctrl+Z / Ctrl+Y)

Full history stack for all canvas operations

F

Lock elements

Lock an element in place so it can't be accidentally moved
